# 🎵 Background Music Setup Guide

## 🚨 **ISSUE: Music Not Working?**

The music button shows a **yellow warning icon** (⚠️) because the `bgm.mp3` file is missing!

## ✅ **Quick Fix:**

1. **Download any MP3 file** you want as background music
2. **Rename it to** `bgm.mp3` (exactly this name)
3. **Place it in** the `public` folder (same location as this file)
4. **Refresh your browser** - the warning icon will become a speaker icon

## 🎼 **Recommended Music Sources:**

### Free & Legal Options:
- **[YouTube Audio Library](https://www.youtube.com/audiolibrary)** - Search for "ambient", "electronic", "instrumental"
- **[Free Music Archive](https://freemusicarchive.org)** - Creative Commons licensed music
- **[Bensound](https://www.bensound.com)** - Free for personal use
- **[Incompetech](https://incompetech.com)** - Kevin MacLeod's royalty-free music

### Quick Test Files:
- Search YouTube Audio Library for "Ambient Electronic" or "Space Music"
- Download any track under 5MB in MP3 format

## 📁 **File Requirements:**
- **Name**: `bgm.mp3` (case-sensitive)
- **Location**: `public/bgm.mp3`
- **Format**: MP3, WAV, OGG, or AAC
- **Size**: Under 5MB recommended
- **Length**: 2-5 minutes (will loop automatically)

## 🎮 **How It Works:**
- Click the speaker icon to play/pause
- Music loops automatically
- Mute button appears when playing
- Volume is set to 30% (adjustable in code)

## 🔧 **Troubleshooting:**
- **Yellow warning icon**: Audio file missing
- **Click doesn't work**: Check browser console for errors
- **No sound**: Check browser audio permissions
- **File not loading**: Ensure file is in `public/` folder and named exactly `bgm.mp3`
